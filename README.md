<p align="center">
  <b>A modern web-based food ordering platform with smooth user experience.</b>

---

## 📖 Overview

**Jigato Food** is a full-stack web application designed to simplify online food ordering.
Users can browse menus, add items to cart, and place orders seamlessly.

---

## ✨ Key Features

* 🛒 **Cart System** – Add/remove items easily
* 🍽️ **Food Browsing** – View available dishes with details
* 📦 **Order Management** – Place and manage orders
* 📧 **Email Integration** – Order notifications via email
* 💻 **Responsive UI** – Works across devices

## ⚙️ Installation & Setup

1. **Clone the repository**

```bash id="9nkc4z"
git clone https://github.com/jigarahir1475/jigato-food.git
```

2. **Move to XAMPP htdocs**

```id="3n5dny"
C:\xampp\htdocs\
```

3. **Start Services**

* Apache ✅
* MySQL ✅

4. **Run in Browser**

```id="7a2k3m"
http://localhost/jigato-food
```
